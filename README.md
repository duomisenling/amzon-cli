- Cherry Studio 安装与更新：[`amz-cli/docs/CHERRY_STUDIO_INSTALL.md`](amz-cli/docs/CHERRY_STUDIO_INSTALL.md)
- CLI 命令手册：[`amz-cli/docs/COMMANDS.md`](amz-cli/docs/COMMANDS.md)
- Agent 长版说明：[`amz-cli/docs/AGENT.md`](amz-cli/docs/AGENT.md)

> 注意：Git 仓库外层目录叫 `amzon-cli`，真正需要交给 Cherry Studio 的工作目录是其中的 `amz-cli` 子目录。
