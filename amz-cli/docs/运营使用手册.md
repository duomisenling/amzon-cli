# 亚马逊运营助手 · 使用手册（给运营同事）

这份手册教你**怎么跟 AI 助手开口**，让它稳定地帮你查数据、做运营。不用懂技术，照着说就行。

---

## 一、开始之前

- 在 Cherry Studio 里打开配好 `amz-cli` 技能的那个助手/智能体，**新开一个会话**。
- 直接用大白话说你要干什么即可，例如「帮我看看店铺最近卖得怎么样」。
- 助手每次会**先只读查询**给你看结果；**改动类操作**（改价、改预算等）它只会先给你预览，最终执行要**你在弹出的审批卡上点允许**。

> 说话小技巧：**一句话里把关键信息给全**——哪个店铺/站点、哪个商品（ASIN 或 SKU）、什么时间范围。信息给全了，它一次就能查对，不用来回问。

---

## 二、查数据（只读，随便问）

下面左边是你想知道的，右边是**照着说**的示范。把示范里的占位符换成你的真实信息。

| 你想知道 | 照着说 |
|---|---|
| 全店最近卖了多少 | 「查一下**美国站**最近 **7 天**的销售额和销量」 |
| 某个商品卖得怎样 | 「看看 ASIN **B0XXXXXXX** 最近 **30 天**的销量」 |
| 某个 SKU 的表现 | 「SKU **ABC-123** 最近 14 天卖了多少」 |
| 最近的订单 | 「列一下最近的订单」／「查订单号 **123-4567890** 的详情」 |
| FBA 还有多少库存 | 「查一下 FBA 库存」 |
| FBA 货件到货情况 | 「看看最近的 FBA 货件有没有收货差异」 |
| 我自己的 listing | 「列出我店铺**美国站**的 listing」／「查 SKU **ABC-123** 的 listing 详情」 |
| 竞品/BuyBox | 「查 ASIN **B0XXXXXXX** 的商品详情和 BuyBox」 |
| 费用预估 | 「估算一下 ASIN **B0XXXXXXX** 的亚马逊费用」 |
| 广告账户/活动 | 「列出我的广告账户」／「列出广告活动」 |
| 差评/卖家反馈 | 「拉一份店铺的卖家反馈报告」（注意：这是全店反馈，不能按单个 ASIN 过滤） |

**关于站点/区域**：默认查的是账号的默认区域。查**欧洲站点**（英德法意西）时，明确带上国家，例如「查**德国站** DE 的…」。北美和欧洲是两套独立凭证，查不到对方是正常的。

---

## 三、做改动（改价、改预算等 —— 要审批）

这些属于**写操作**，助手不会直接改，而是走「**先预览 → 你审批 → 才执行**」。

**怎么开口：**

| 你想做 | 照着说 |
|---|---|
| 改 listing（标题/价格/图片等） | 「把 SKU **ABC-123** 的**标题**改成『…』」 |
| 改广告日预算 | 「把广告活动 **XXXX** 的日预算改成 **20**」 |
| 启用/暂停广告 | 「暂停广告活动 **XXXX**」／「启用广告活动 **XXXX**」 |
| 改关键词竞价 | 「把关键词 **XXXX** 的竞价改成 **0.8**」 |
| 加否定词 | 「给广告组 **XXXX** 加否定词『免费』」 |
| 从零建一套关键词广告 | 「用这些关键词给 ASIN **B0XXX** 建一套广告：…（词、预算、竞价）」 |

**执行流程（重要，务必照做）：**

1. 助手先给你一份**预览**：改什么、从什么值变成什么值。
2. **你核对**：站点、商品、金额、关键词对不对。
3. 对了，就等 Cherry **弹出审批卡**，在卡上**点允许**——这一步才真正写进亚马逊。
4. 卡上的内容和预览**对不上就点拒绝**，让助手重新预览。

> ⚠️ 安全底线：聊天里打「确定/好/Y」**不算**批准，只有**审批卡上的点击**才算。金额、商品、站点和预览不一致，一律拒绝。

---

## 四、让它更听话的小技巧

- **信息一次给全**：ASIN/SKU、站点、时间范围、金额一起说，减少来回。
- **不满意就直接纠正**：「不对，我要的是**德国站**」「时间改成最近 **90 天**」——它会重来。
- **只要结论**：它默认会用中文总结金额、时间、异常项，不用你看原始数据。想看明细再说「把明细也列出来」。
- **一次只说一件事**：复杂需求拆开说，别把五六件事堆一句。

---

## 五、遇到"不太对劲"怎么办

| 现象 | 多半是什么 | 你可以怎么做 |
|---|---|---|
| 它说找不到 `amz-cli` | 没装好或没重装 | 让管理员重跑安装命令 |
| 改动类操作没弹审批卡 | 写工具（MCP）没挂上 | 找管理员确认 Cherry 里挂了 MCP |
| 查欧洲站点查不到 | 用的是北美默认区域 | 明确说「查德国站 DE」，或让管理员确认配了欧洲凭证 |
| 报"没有凭证/refresh token" | 这台电脑凭证没配好 | 找管理员配 `~/.amz-cli/.env` 并**重启 Cherry** |
| 改了凭证还是不生效 | 改完没重启 | 让管理员**重启 Cherry**（配置只在启动时读一次） |
| 它跑一堆没用的命令 | 助手在瞎试 | 把需求说得更具体；或让管理员升级到最新版 |

**任何看不懂的红色报错**，不用自己猜——**截图发给管理员**，里面通常写清了缺什么。

---

## 六、一句话记住

> 说清楚「**哪个站点 · 哪个商品 · 什么时间 · 想干什么**」，查数据随便问，改动看预览、在**审批卡**上把关。
